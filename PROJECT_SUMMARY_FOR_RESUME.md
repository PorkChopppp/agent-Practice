# AI研究助手项目总结

## 项目概述

AI研究助手是一个基于人工智能的自动化研究工具，能够根据用户提供的主题自动收集相关信息并生成结构化研究报告。该项目整合了多种现代技术，包括向量数据库、大语言模型和微服务架构，展现了我在全栈开发和AI应用方面的技能。

## 核心功能

- 自动研究指定主题：根据用户输入的主题自动收集相关信息
- 生成结构化研究报告：使用大语言模型生成专业的研究报告
- 向量数据库集成：集成 Milvus 向量数据库进行高效相似性搜索
- 关系数据库存储：使用 PostgreSQL 存储最终报告和元数据
- 双AI代理架构：研究员代理和作家代理协同工作
- 双Web框架支持：同时支持 Flask 和 FastAPI 两种 Web 框架
- 异步处理：支持后台任务处理，提高用户体验

## 技术栈

- **语言**: Python 3.8+
- **AI框架**: LangChain, LangGraph
- **大语言模型**: OpenAI GPT, SiliconFlow 兼容模型
- **向量数据库**: Milvus
- **关系数据库**: PostgreSQL
- **容器化**: Docker
- **Web框架**: Flask/FastAPI
- **前端**: HTML, CSS, JavaScript

## 系统架构

项目采用模块化设计，主要包括以下组件：

1. **AI代理模块**：包含研究员代理、作家代理、协调者代理等多个AI代理，分别负责不同的任务
2. **数据库模块**：实现了Milvus向量数据库和PostgreSQL关系数据库的客户端
3. **工作流模块**：定义了从研究到报告生成的完整自动化流程
4. **Web接口**：提供了Flask和FastAPI两种Web框架的实现
5. **配置管理**：统一管理系统配置和环境变量
